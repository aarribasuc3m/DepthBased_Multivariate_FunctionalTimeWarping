# Depth-based estimation in the latent deformation model
THESE ARE THE FILES ACCOMPANYING THE PAPER: "Depth-based estimation for time-warping
multivariate functional model"

List of files:

1. DB_est.R: main R function to conduct depth-based estimation in the latent deformation model

2. DB_est_functions.R: auxiliary R functions for depth-based estimation

3. example_visualization.R: Example of data generation, estimation and visualization

4. sim_fun.R: R functions to simulate data in the latent deformation model

5. aux_fun.R: auxiliary R functions for plots

6. WHyRA.R: R code to create the WHyRA plot from warping estimates

7. simulation_study.R: R code to replicate the simulation study of the article and the computation times comparison

8. figures_paper_simulation.R: R code to reproduce the figures in the paper

9. est_comp.R: NOT PROVIDED. R code implementing the estimation approach of Carroll and Muller (2023). To be downloaded from: 
https://academic.oup.com/biometrics/article/79/4/3345/7585581?login=false#supplementary-data
